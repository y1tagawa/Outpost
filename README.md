# Outpost

## 概要

サンプルアプリ置き場です。

- UnityProjecsts
    - HelloCardboard
        - Google Cardboard for Unityを使用したサンプルアプリです。

- FlutterProjects
    - Flutterを使用したサンプルアプリです。
    - hello_flutter
        - CustomPainterを使用してアニメーションを描画しています。

    - hello_flutter_db
        - アプリにバンドルしたデータベースを使用しています。
    
    - hello_flutter_http
        - HTTPSによりサーバと通信を行います。

ライセンスは各プロジェクトのLICENSEファイルを参照してください。
